Cloud Foundry Training

EngineerBetter deliver on-site, hands-on training for Cloud Foundry

Training

We provide on-site hands-on, instructor-led Cloud Foundry training. After running training at Cloud Foundry Summit the Cloud Foundry Foundation asked us to build the official training course for the Cloud Foundry Certified Developer exam.

We offer two levels of course. The aim of both of them is to establish a robust and accurate mental model of Cloud Foundry, by experiencing how to use the platform through practical exercises. Students are not given step-by-step instructions; instead the practical exercises lead them to discover solutions through use of the CF CLI and the CF documentation.

Zero To Hero: Cloud Foundry Beginners Course

  • Duration: 1 day
  • Price per student: £650
  • Class size: Minimum 10, Maximum 20
  • Outcomes: Able to use all core Cloud Foundry features as a developer, such as pushing apps, using multiple buildpacks, using services, 12 Factor Application design, debugging applications, use of routing for zero-downtime deploys.
  • Student Pre-requisites: Little to no CF experience; experience of writing/deploying web applications; comfortable using the command line.
  • Technical Pre-requisites: 64 bit Linux, macOS or Windows; 1GiB RAM; public web access; text editor of choice.

Cloud Foundry Certified Developer Training

This is the premium Cloud Foundry training experience and was developed by our team alongside the Certified Developer program. The course aims to leave students completely self sufficient in using Cloud Foundry, by the end of the 4 days.

  • Duration: 4 days
  • Price per student: £2,600
  • Class size: Minimum 8, Maximum 18
  • Outcomes: Exposure to all topics in the Cloud Foundry Certified Developer curriculum, including all main capabilities of the platform, use cases and usage patterns, and 'under the hood' implementation details. Syllabus available on request.
  • Student Prerequisites: Experience of the whole application development lifecycle; self-motivated and able to reason about problems; comfortable using the command line; comfortable modifying Spring/Java applications.
  • Technical Pre-requisites: 64 bit Linux, macOS or Windows; 1GiB RAM; public web access; Git client installed and ready to use; text editor of choice.

For either course we provide one instructor for every 10 students, so for example if you have a class size of 15 this would need two instructors. Travel and expenses (outside of London, England) are charged separately.
